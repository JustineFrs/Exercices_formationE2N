Exercices PARTIE #1 MySQL

Exo1

Créer une base de données appelée competences.

mysql -h localhost -u root -p CREATE DATABASE nom_base; GRANT ALL PRIVILEGES ON nom_base.* TO 'jufrs'@'localhost' IDENTIFIED BY 'MDP'; quit mysql -u jufrs -p mdp USE nom_base

Exo2

Dans la base de données competences, créer une table langages comprenant deux champs : id : type numérique, clé primaire et auto-incrémenté. langage : type alphanumérique.

CREATE TABLE `langage`( `id` INT(11) UNSIGNED NOT NULL AUTO_INCREMENT, `langages` VARCHAR(50) NOT NULL, PRIMARY KEY(id) ) ENGINE=INNODB;

Exo3

Créer une base de données appelée colysee avec l'encodage UTF-8.

mysql -h localhost -u root -p --default-character-set=utf8 CREATE DATABASE `colysee`; GRANT ALL PRIVILEGES ON `colysee`.* TO 'jufrs'@'localhost' IDENTIFIED BY 'MDP'; quit mysql -u jufrs -p mdp USE nom_base

Exo4

Dans la base colysee, créer la table clients contenant 5 champs : id, clé primaire, auto-incrémenté nom prenom carte_fidelite reduction

CREATE TABLE `clients`( id INT(11) UNSIGNED NOT NULL AUTO_INCREMENT, nom VARCHAR(50) NOT NULL, prenom VARCHAR(50) NOT NULL, carte_fidelite TINYINT(1), reduction DECIMAL(3,2), PRIMARY KEY(id) ) ENGINE=INNODB;

Exo5

Dans la base colysee, créer la table salles avec deux champs : id, clé primaire, auto-incrémenté nombre_sieges

CREATE TABLE `salles`( id INT(11) UNSIGNED NOT NULL AUTO_INCREMENT, nombre_sieges INT(5) NOT NULL, PRIMARY KEY(id) ) ENGINE=INNODB;

Exo6

Dans la base colysee, créer la table spectacles avec les champs : id, clé primaire, auto-incrémenté intitule artiste type genre date heure duree salles_id

CREATE TABLE `spectacles`( id INT(11) UNSIGNED NOT NULL AUTO_INCREMENT, intitule VARCHAR(50) NOT NULL, artiste VARCHAR(50) NOT NULL, type VARCHAR(50) NOT NULL, genre VARCHAR(50) NOT NULL, date DATE NOT NULL, heure TIME NOT NULL, duree TIME NOT NULL, salle_id INT(11) NOT NULL, PRIMARY KEY(id) ) ENGINE=INNODB;

Exo7

Dans la base colysee, créer la table reservations avec les champs : id, clé primaire, auto-incrémenté spectacles_id clients_id nombre_billets

CREATE TABLE reservation( id INT(11) UNSIGNED NOT NULL AUTO_INCREMENT, `spectaclesId` INT(11) UNSIGNED NOT NULL, `clientsId` VARCHAR(50) NOT NULL, `nombreBillets` INT(2) NOT NULL, PRIMARY KEY(id) ) ENGINE=INNODB;
